
# fsociety center

Script Python para Termux/Linux/Windows.
Automatiza tarefas relacionadas à fsociety.
Inclui funcionalidades de rede e sistema.
Executa comandos de forma interativa.
Permite manipulação de arquivos locais.
Possui funções de logging simples.
Preparado para estudo e aprendizado.
Não realiza ações maliciosas por padrão.
Compatível com múltiplas plataformas.
Fácil de personalizar e expandir.
Recomendado para testes em laboratório seguro.

## Feature
- **Tool Installation**  
Install Single Tool  
`lzmx > set_install 1`  
Install Multi Tool  
`lzmx > set_install 1 2 3 4`  
Install All Tool  
`lzmx > set_install @`  
- **Default Dir Install**
On `lazymux.conf` replace symbol ~ with directory you want  
Example: lazymux.conf  
`HOME = /sdcard`
